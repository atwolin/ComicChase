# ComicChase UI

歡迎來到 ComicChase UI！這是 ComicChase 專案的前端部分，提供一個現代化、響應式且使用者友善的介面，讓使用者可以搜尋、追蹤和管理他們喜愛的漫畫。

## ✨ 功能特色

- **使用者認證**：安全的註冊和登入功能。
- **漫畫搜尋**：強大的搜尋功能，可以按標題、作者或類型尋找漫畫。
- **個人收藏**：將漫畫加入您的個人收藏並進行追蹤。
- **出版進度**：追蹤漫畫的最新更新
- **響應式設計**：在桌面和行動裝置上皆有良好的瀏覽體驗。

## 🚀 開始使用

請遵循以下說明，在您的本機環境中設定並執行此專案。

### 先決條件

請確保您的系統已安裝以下軟體：

- [Node.js](https://nodejs.org/) (建議使用 v18.x 或更高版本)
- [npm](https://www.npmjs.com/) 或 [yarn](https://yarnpkg.com/)

### 安裝

1.  複製此儲存庫到您的本機電腦：
    ```bash
    git clone https://github.com/your-username/ComicChase.git
    ```
2.  進入 UI 目錄：
    ```bash
    cd ComicChase/ui
    ```
3.  安裝專案依賴：
    ```bash
    npm install
    # 或如果您使用 yarn
    # yarn install
    ```

## 🏃‍♂️ 執行專案

安裝完成後，您可以執行以下指令來啟動開發伺服器：

```bash
npm run dev
# 或如果您使用 yarn
# yarn dev
```

應用程式將會在 `http://localhost:3000` 上執行。在瀏覽器中開啟此網址即可看到應用程式。
